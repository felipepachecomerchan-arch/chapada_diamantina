🌿 Publicar Roteiro no GitHub Pages

Seu link ficará público e abrirá direto no Safari — sem WebView do WhatsApp

⏱ Cerca de 5 minutos · Gratuito

📁 Arquivos para fazer upload

📄chapada_roteiro.htmlpágina principal
⚙️manifest.jsonconfiguração PWA
🔧sw.jscache offline
🖼️icon-192.pngícone pequeno
🖼️icon-512.pngícone grande
1
Crie uma conta gratuita no GitHub
Acesse github.com e clique em Sign up. Use qualquer e-mail — é totalmente gratuito.
2
Crie um repositório novo
Após entrar, clique no botão + (canto superior direito) → New repository.

Preencha assim:
• Repository name: chapada-2026
• Marque Public
• Marque ✅ Add a README file
• Clique em Create repository
3
Faça upload dos 5 arquivos
Dentro do repositório criado, clique em Add file → Upload files.

Arraste ou selecione os 5 arquivos listados acima de uma vez.
Clique em Commit changes para salvar.
💡 Certifique-se que o arquivo principal se chama exatamente chapada_roteiro.html
4
Ative o GitHub Pages
No repositório, clique em Settings (engrenagem) → no menu lateral, Pages.

Em Source, selecione:
• Branch: main
• Pasta: / (root)

Clique em Save.
⏳ Aguarde 1–2 minutos para o site ficar disponível.
5
Copie o link e compartilhe!
Ainda em Settings → Pages, aparecerá o link:
🔗 https://SEU-USUARIO.github.io/chapada-2026/chapada_roteiro.html

Mande esse link pelo WhatsApp. Ao clicar, o iPhone abre direto no Safari — com mapa, accordion e checkboxes funcionando perfeitamente! ✅

✅ Como ficará o seu link

https://SEU-USUARIO.github.io/chapada-2026/chapada_roteiro.html

Links https:// nunca são interceptados pelo WhatsApp — abrem sempre no Safari.

🤔 Por que isso funciona?

🔗O WhatsApp intercepta apenas arquivos locais (.html). Links https:// são sempre abertos no navegador padrão do iPhone (Safari).
🆓GitHub Pages é completamente gratuito para projetos públicos, sem limite de visitas.
📲Como o roteiro é uma PWA, quem abrir o link no Safari pode "Adicionar à Tela de Início" e usar como um app, inclusive offline.
🔒O repositório pode ser público sem problema — o HTML não contém dados pessoais sensíveis.
